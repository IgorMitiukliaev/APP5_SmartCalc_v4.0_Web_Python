Calculator

The application works in 2 modes:

  1. Basic calculations. You can type an expression using screen or hardware keyboard, then press " = " virtual button to get a result. Successful calculations are stored in the app history.
  2. Formula calculation. If an expression contains "x" variable, you can either type " x = " following the value right after the expression, or just press " = ". In that case you will be prompted to enter value.

"/ - /" button is a shortcut to wrap the selected part of the expression with "(- ... )".

"C" clears the bottom line on the display, "BS" clears the last symbol or function name, if the line ends with it.

To get the expression chart, simply click "graph".

To view calculation history, click "hist" button. After that you will be able to clear it completely.
